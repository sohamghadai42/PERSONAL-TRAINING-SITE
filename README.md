# Personal Training Website

A responsive personal training website built with HTML and CSS. This is my first frontend project, deployed and hosted on Netlify.

##  Live Demo

[View Live Site](https://personaltrainingapp.netlify.app/)

##  About The Project

This project is a personal training gym website that showcases services, coaching programs, and community features. It was built as a learning project to practice fundamental web development skills.

### Design Credit
- Design: [Figma Community](https://www.figma.com/community)
- Implementation: Custom HTML & CSS

##  Built With

- HTML5
- CSS3
- Deployed on Netlify

##  Features

- Clean and modern design
- Responsive layout (optimized for mobile and laptop screens)
- Multiple pages:
  - Home page
  - About page
  - Reserve Your Spot page
- Contact information section
- Social media links

##  Getting Started

### Prerequisites

No special prerequisites needed! Just a web browser.

### Installation

1. Clone the repository
```bash
   git clone https://github.com/yourusername/personal-training-website.git
```

2. Navigate to the project directory
```bash
   cd personal-training-website
```

3. Open `index.html` in your browser
```bash
   # On Mac
   open index.html
   
   # On Windows
   start index.html
   
   # On Linux
   xdg-open index.html
```
